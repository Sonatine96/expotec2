En Laboratorio de Aplicaciones aprendemos principalmente a usar programas de la computadora.

- Aprendemos a usar Excel. Sirve para organizar datos mediante hojas de cálculo.

- Aprendemos a usar Word. Word es un procesador de texto que permite crear, editar y dar formato a documentos de texto como cartas, informes, trabajos académicos y currículums, permitiendo también insertar elementos como imágenes, gráficos y tablas.

- Aprendemos a usar Powerpoint. Sirve para presentar información de forma visual y atractiva usando texto, imágenes, gráficos y videos. Se utiliza para fines educativos, comerciales o personales, ya sea para presentar ideas, marketing de productos, enseñar en clases o dar discursos, etc.
